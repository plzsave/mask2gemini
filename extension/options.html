マスクしない語句（ホワイトリスト）

ここに登録した語句は自動マスクの対象から外れます（自動ルールより優先）。 自社名や、出しても問題ない自分の名前などを登録してください。 照合は空白を無視します（「山田 太郎」は「山田太郎」にも一致）。

    チーム共有: エクスポートした JSON を共有リポジトリや共有ドライブに置き、 各メンバーがインポートしてください（既存の登録に追記されます）。

    エンジニア向けオプション

    オンにすると、確認画面に「ワイヤーフレームを保存」ボタンが表示されます。 マスク済みの画面構造を Excalidraw で編集できる形式で書き出します （マスクした文字列はファイルに含まれません。ページ構造を読めた場合のみ利用可）。

    オンにすると、確認画面に「デバッグ表示」チェックボックスが現れます。 マスク判定の理由（reason）を枠色で可視化したり、判定ログを JSON でコピーしたりできます。
